事情是这样的，彼时，本人于某游戏中，征战四方，驰骋沙场。然而天有不测风云，忽闻“zz，下午要交的实验报告你打印了没？”  
下午！？这怎么可能？C#课不是明天的吗？  
“啊，可是，好像有领导要检查学生的实验报告。”  
生死时速，千钧一发，本人存了个档（顺便先把邻国的宣称伪造了），马上开写——  
记事本，好，简单，有手就行，但如何写出本人既往的独特风格？  
想到当年正火的...一个遭人唾骂的游戏主播，好，记事本的名字就叫抽象圣经吧！  
于是，这本神秘而儒雅的抽象圣经，降世了。
